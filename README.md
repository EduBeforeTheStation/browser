## Description

Privacy protection browser to prevent crime damage on the web.

<br/>

## Installation

```bash
$ git clone https://github.com/krypton-browser/krypton.git
$ cd krypton
$ yarn
```

<br/>

## Running the app

```bash
$ yarn start
```

<br/>

## Build

```bash
# all
$ yarn build

# main
$ yarn build:main

# renderer
$ yarn build:renderer
```
